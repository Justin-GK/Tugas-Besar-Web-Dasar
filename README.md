### 🛠️ Spesifikasi & Fitur yang Diimplementasikan
Sesuai dengan ketentuan tugas, website ini telah memenuhi **15 fitur wajib** dan spesifikasi teknis sebagai berikut:

#### 1. Desain & Layout (CSS)
- [x] **Mobile First Design**: Tampilan dioptimalkan untuk perangkat mobile terlebih dahulu.
- [x] **Media Queries**: Layout responsif yang menyesuaikan otomatis saat dibuka di Desktop/Tablet.
- [x] **CSS Flexbox**: Digunakan pada komponen Navbar, Hero Section, dan Footer.
- [x] **CSS Grid**: Digunakan pada galeri Arsitektur & Pakaian Adat untuk tata letak yang rapi.
- [x] **Cross Browser**: Desain konsisten di Chrome, Firefox, Safari, dan Edge.
- [x] **External CSS**: Seluruh gaya desain dipisah ke file `style.css`.

#### 2. Komponen Visual & Interaktivitas
- [x] **List**: Digunakan pada daftar kuliner dan kesenian khas.
- [x] **Image**: Galeri foto budaya dengan optimasi `object-fit`.
- [x] **Dropdown**: Navigasi menu bertingkat pada header.
- [x] **Transitions**: Efek halus saat kursor mengarah ke tombol (hover).
- [x] **Animations**: Animasi *fade-in* dan *slide-up* saat halaman pertama kali dimuat.
- [x] **Text Effect**: Penggunaan bayangan teks (`text-shadow`) pada judul utama.
- [x] **Buttons**: Tombol interaktif untuk navigasi dan aksi form.

#### 3. Logika Pemrograman (JavaScript)
- [x] **External JS**: Seluruh logika dipisah ke file `script.js`.
- [x] **Form**: Input buku tamu dan kuis interaktif.
- [x] **JS Functions**: Fungsi untuk validasi pesan dan penghitungan skor kuis.
- [x] **JS Events**: Penggunaan `onclick` untuk memproses input pengguna.
- [x] **JS Conditional**: Logika `if-else` untuk validasi form dan penentuan hasil kuis.
- [x] **JS Looping**: Penggunaan `for` loop untuk memproses array jawaban pada kuis.
